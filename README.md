# Airport Data Analysis Project

## Project Overview

This project analyzes airport and airline operational data to identify trends in passenger traffic, airline performance, busiest routes, and geographic aviation activity.

The project was developed using:

* Python
* SQL
* Power BI

## Tools Used

* Python
* Pandas
* Matplotlib
* MySQL
* Power BI

## Key Analysis Performed

* Top airports by passenger movement
* Top airlines by passenger traffic
* Busiest flight routes
* Longest routes
* Top states by passenger traffic
* Region traffic analysis

## Dashboard Features

* KPI Cards
* Airport Analysis
* Airline Analysis
* Route Analysis
* Geographic Analysis

## Project Files

* Jupyter Notebook
* SQL Queries
* Power BI Dashboard
* Cleaned Dataset

## Conclusion

This project demonstrates practical applications of data cleaning, exploratory data analysis, SQL querying, data visualization, and dashboard development using aviation operational data.
